описание формата файла модели:

//начало файла
[секция параметров]
[секция помещений]
[секция перегородок]
[секция источников шума]
[секция расчётных точек]
//конец файла

секция параметров:
  groundID(int32) - id покрытия под рельсами в БД
  speed(double) - скорость вагона
  RailMaterialId(int32) - id материала, из которого изготовлены рельсы

секция помещений:
  rooms count(int32) - количество помещений
  далее rooms count раз:
    l(int32) - длина UTF8-представления названия помещения
    b(byte[]) - UTF-8-представление названия помещения

секция перегородок:
  borders count(int32) - количество перегородок
  далее borders count раз:
    l(int32) - длина UTF8-представления названия перегородки
    b(byte[]) - UTF-8-представление названия перегородки
    room1index(int32) - номер помещения 1, если такое назначено перегородке;-1 в противном случае
    room2index(int32) - номер помещения 2, если такое назначено перегородке;-1 в противном случае
    OnTopX(double) - x-координата точки на верхнем ребре прямоугольной перегородки
    OnTopY(double) - y-координата точки на верхнем ребре прямоугольной перегородки
    OnTopZ(double) - z-координата точки на верхнем ребре прямоугольной перегородки
    BottomLeftX(double) - x-координата левой нижней точки прямоугольной перегородки
    BottomLeftY(double) - y-координата левой нижней точки прямоугольной перегородки
    BottomLeftZ(double) - z-координата левой нижней точки прямоугольной перегородки
    BottomRightX(double) - x-координата правой нижней точки прямоугольной перегородки
    BottomRightY(double) - y-координата правой нижней точки прямоугольной перегородки
    BottomRightZ(double) - z-координата правой нижней точки прямоугольной перегородки
    layersCount(int32) - количество слоёв перегородки
    далее layersCount раз:
      layerMaterialId(int32) - id материала слоя (в БД)
      layerThickness(double) - толщина слоя(в сантиметрах)

секция источников шума:
  sources count(int32) - количество источников
  далее sources count раз:
    l(int32) - длина UTF8-представления названия источника шума
    b(byte[]) - UTF-8-представление названия источника шума
    roomIndex(int32) - номер помещения, в котором установлен источник шума, если оно назначено источнику;-1 в противном случае
    PosX(double) - x-координата точки, в которой расположен источник шума
    PosY(double) - y-координата точки, в которой расположен источник шума
    PosZ(double) - z-координата точки, в которой расположен источник шума
    pow_lvl315(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=31,5 Hz
    pow_lvl63(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=63 Hz
    pow_lvl125(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=125 Hz
    pow_lvl250(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=250 Hz
    pow_lvl500(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=500 Hz
    pow_lvl1000(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=1000 Hz
    pow_lvl2000(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=2000 Hz
    pow_lvl4000(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=4000 Hz
    pow_lvl8000(double) - уровень мощности источника в октавной полосе со среднегеометрической частотой f=8000 Hz

секция расчётных точек:
  calcpoint count(int32) - количество точек
  далее calcpoint count раз:
    l(int32) - длина UTF8-представления названия расчётной точки
    b(byte[]) - UTF-8-представление названия расчётной точки
    roomIndex(int32) - номер помещения, в котором установлен источник шума, если такое назначено расчётной точке;-1 в противном случае
    PosX(double) - x-координата точки, в которой расположена расчётная точка
    PosY(double) - y-координата точки, в которой расположена расчётная точка
    PosZ(double) - z-координата точки, в которой расположена расчётная точка
